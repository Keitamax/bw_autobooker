This tool automates the booking for BoulderWorld.

1. Ensure you have Chrome v.88 or 89 installed
2. Modify example.json file to fit your data
3. Run the jar: java -jav ./boulderworld_autobooker
4. Copy content of autobooker_drivers to requested path and hit enter
5. Follow the instructions

The tool will try to find your slot.
If it doesn't find it, it will refresh and try again (as long as you don't stop it)
